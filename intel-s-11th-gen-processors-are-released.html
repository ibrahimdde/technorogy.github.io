Intel's 11th gen processors are released

03/30/21
Months after the teaser, the processors are finally out. There are several changes, only change that stands up is PCI-e 4.0 standart will be supported in the newest line-up. Some clock speeds are up, other than that it is very similar to the old gen processors.

One notable thing is high-end processors finally received a integrated graphics unit that can compete with AMD Ryzen's G line-up. Gaming performance of them will depend of the variables like RAM, cooling and the game itself. But it is safe to say that good performance should not be expected from them. They surely can run older games or competitive games okay-ish, that is for sure.

Intel still use 14 nm build, which is inferior to it's main competetor's 7 nm build.
